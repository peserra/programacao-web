Plano Alimentar - Nutrição
Paciente: _________________________________________________________
Metas:
	•	Avaliar o valor nutricional dos alimentos consumidos;
	•	Avaliar os ingredientes X modo de preparo dos alimentos consumidos (caseiro e industrializado);
	•	Ter um planejamento alimentar / Identificar alimentos práticos para o dia-a-dia;
	•	Ter atenção X porção;
	•	Ter atenção na fome (emocional / fisiológica);
	•	Identificar os alimentos que favorecem maior saciedade (fome fisiológica);
	•	Identificar substitutos para o doce (tradicional): alimentos com valor nutricional, paladar adocicado e sem açúcar refinado;
	•	Adoçar os alimentos de forma indireta / Treinar para tolerar os alimentos mais amargos / azedos (processo de redução / Desmame do açúcar / adoçante);
	•	Observar a presença da proteína nas principais refeições.
ALIMENTO DIÁRIO / HABITUAL
	•	Azeite extra virgem (frio / cru) = 2 colheres de chá (1 colher no Almoço + 1 colher no Jantar);
	•	Fruta = 2-3 porções de fruta por dia / 1 porção de cada vez = 1 fruta unitária ou 1 xícara de chá;
	•	Verdura e/ou legumes do grupo do verde (almoço + jantar) = à vontade ou 1/2 prato grande;
	•	Farelo de aveia / Quinoa / Amaranto / Sementes (Chia, Gergelim, Linhaça);
	•	Iogurte desnatado ou semidesnatado e sem açúcar / Natural = 1-2 potes (170ml);
	•	Ovo / Clara de ovo = até 3 ovos inteiros no dia / + somente a clara do ovo;
	•	Água = 1,5L / dia: com estratégia;
	•	Oleaginosas (Castanha do Pará, Castanha de Caju, Nozes, Amêndoa, Avelã, Pistache, Macadamia, Amendoim).
OBSERVAÇÕES:
	•	Observar o consumo do carboidrato (Índice Glicêmico X fibra);
	•	Observar os alimentos consumidos: Quantidade / Quantidade / Modo de preparo;
	•	Ter estratégia na escolha do alimento do horário.

EXEMPLO DE PLANO ALIMENTAR
CAFÉ DA MANHÃ = Carboidrato + Proteína.
Proteína:
	•	Leite (desnatado ou semidesnatado) = 180 ml;
	•	Ou Iogurte (desnatado ou semidesnatado / Natural / sem açúcar) = 170 ml;
	•	Ou Ovo inteiro (1-2 unidades);
	•	Ou Queijo magro (Ricota / Cottage / Frescal / Branco / Mussarela de Búfala) = 2 fatias / largura de 1 dedo.
Carboidrato:
	•	1ª Opção: Legume amarelo e/ou laranja (+ batata doce / mandioca / batata Yacon / milho) / Farelo de Aveia / Quinoa / Amaranto / Panqueca de aveia / Vitamina / Bolo de banana / Suco fortificado;
	•	2ª Opção: Panqueca de tapioca / Cereal integral X Granola sem açúcar / Pão integral 100% / Pão de aveia / Cuscuz com fibra;
	•	3ª Opção: Tapioca com fibra / Torrada integral / Torrada Multigrãos / Biscoito integral / Biscoito Multigrãos / Biscoito de Gergelim.
Recheio (se necessário):
	•	Azeite extra virgem / Ghee / Manteiga / Manteiga de azeite / Cream Cheese / Requeijão / Geleia sem açúcar / Creme de Oleaginosa sem açúcar / Patê de legumes / Patê de abacate / Patê de leguminosa: até 1 colher de chá.
Observações:
	•	Procurar não adoçar nada;
	•	Se necessário adoçar os alimentos de forma indireta;
	•	Poderá preparar suco fortificado com couve + 1 porção de fruta + água + Farelo de aveia e/ou Quinoa e/ou Amaranto e/ou Gérmen de trigo e/ou sementes;
	•	Granola: Sempre utilizar para dar crocância, preferir o consumo do Farelo de aveia e/ou Quinoa e/ou Amaranto e/ou Gérmen de trigo;
	•	Poderá adicionar ervas frescas ou desidratadas no ovo / queijo;
	•	Poderá adicionar canela e/ou cacau em pó na fruta / mingau / panqueca.

LANCHE DA MANHÃ (Se acordar cedo / Se necessário)
1ª opção de escolha:
	•	Fruta / Papa de fruta / Legume amarelo / laranja / Pipoca / Gelatina caseira / Geladinho caseiro / Ovo / Clara de ovo / Ovo de codorna / Omelete / Leite / Iogurte / Queijo magro / Oleaginosas / Suco fortificado.
2ª opção de escolha:
	•	Suco diluído / Água de coco / Bananinha sem açúcar / Fruta seca / Clips de fruta.
Observações:
	•	Poderá consumir chá / café;
	•	Procurar não adoçar nada;
	•	Se necessário adoçar os alimentos de forma indireta;
	•	Poderá adicionar canela na fruta;
	•	Poderá adicionar ervas frescas ou desidratadas no ovo / queijo.

ALMOÇO = Carboidrato + Proteína + fibra (grupo do verde).
Carboidrato:
	•	4 colheres de sopa do grupo / massa (macarrão) = ½ prato;
	•	Opções: Arroz (+ integral / Multigrãos) e/ou batata (+ batata doce / batata Yacon) e/ou cenoura e/ou mandioca e/ou mandioquinha e/ou inhame e/ou milho e/ou beterraba e/ou abóbora e/ou tomate e/ou macarrão (preferir a massa integral ou com legumes).
Proteína Vegetal:
	•	Leguminosas (feijão, lentilha, ervilha, grão-de-bico) = 4 colheres de sopa (com grão) / sem embutidos = pelo menos 3x por semana.
Proteína Animal:
	•	Peixe = 1 filé do tamanho da mão ou 4 colheres de sopa;
	•	Ou Frango / sem pele = 1 filé do tamanho da mão ou 4 colheres de sopa;
	•	Ou Carne vermelha / sem gordura aparente = 1 filé (fino) do tamanho da mão ou 3 colheres de sopa;
	•	Ou Carne de porco (lombo, bisteca ou pernil) = 1 filé do tamanho da palma da mão (fino) ou 3 colheres de sopa rasas;
	•	Ou Ovo inteiro 1 unidade / + somente a clara do ovo.
Fibras / grupo do verde:
	•	Legumes / verduras = folhas (cru, cozido, ao forno, à vapor: sem ou com pouca adição de gordura, no máximo 1 colher de chá): 1/2 prato grande.
Líquido:
	•	NADA ou água ou água saborizada (180ml).
Observações:
	•	A refeição completa não deve ser ingerida sem fibras = sempre consumir junto com verduras e/ou legumes;
	•	Carboidrato = Procurar distribuir o consumo indicado de 4 colheres de sopa do grupo / preferir sempre o consumo dos carboidratos de menor Índice glicêmico;
	•	Observar o modo de preparo dos alimentos.

LANCHE DA TARDE
1ª opção de escolha:
	•	Fruta / Papa de fruta / Legume amarelo / laranja / Pipoca / Gelatina caseira / Geladinho caseiro / Ovo / Clara de ovo / Ovo de codorna / Omelete / Leite / Iogurte / Queijo magro / Oleaginosas / Suco fortificado / Almôndega assada / Kibe de forno / Bolinho de arroz de forno / Vitamina / Panqueca / Bolo de banana / Mingau de aveia.
2ª opção de escolha:
	•	Suco diluído / Água de coco / Bananinha sem açúcar / Fruta seca / Clips de fruta.
Observações:
	•	Poderá consumir chá ou café;
	•	Procurar não adoçar nada;
	•	Se necessário adoçar os alimentos de forma indireta;
	•	Poderá adicionar canela e/ou cacau em pó na fruta;
	•	Poderá adicionar sementes / aveia / quinoa / amaranto / gérmen de trigo na fruta / suco / iogurte / leite = 1 colher de chá;
	•	Na vontade de comer doce, verificar as melhores opções (alimentos com valor nutricional / com paladar adocicado / sem açúcar).

JANTAR = Carboidrato + Proteína + fibra.
Idem almoço = com Carboidrato (3-4 colheres de sopa) + proteína (carnes de boa digestão ou ovo) + fibra (grupo do verde).
Observações:
	•	Observar o consumo dos grupos;
	•	Poderá não fazer o consumo do arroz e feijão / leguminosa;
	•	Respeitar a presença do grupo do verde.
Ou
Sopa caseira = Carboidrato (legume amarelo e/ou laranja) + Verdura e/ou Legumes (grupo do verde) + Proteína (Leguminosa e/ou carne / frango) = 2 pratos fundos / Batida ou não!
Observações:
	•	Não preparar sopa somente com os ingredientes amarelos / laranja (legumes do grupo do carboidrato);
	•	Preferir sopa caldo / evitar sopa creme;
	•	Poderá adicionar Sementes / aveia / quinoa / amaranto / gérmen de trigo = até 1 colher de sobremesa rasa / principalmente no caso de fome.
Ou
Lanche (+ sábado) = 1 ou 1 1/2 porção / sempre com fibras (vegetais / grupo do verde) -- lanche no prato se possível.
Observações:
	•	Evitar molhos;
	•	Avaliar o habitual da massa / pão.
